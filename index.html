Organizing the Frontend

Results

Fast Deployments
Static bundles are packaged into version directories on S3 buckets.
Releases and roll-backs are instant and require no CDN invalidations.

Scalable
Each project gets its own repository.
Based on standard NPM commands so not coupled to any framework if not needed.

Testing
Projects can be auditted independently.
Local unit tests per projects (JEST)
End to End tests run per project (cypress.io with offline mocked APIs)
Broad integration tests (Selenium) on staging servers

Results

Performance
The core library is cached accross all products for faster page load.
Combination of Webpack code splitting and external dependencies.

Security
Core team is responsible for managing and whitelisting npm packages that are available to projects.
